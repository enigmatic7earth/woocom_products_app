E-Commerce app using Flutter, following articles by ![alt text](https://cdn-images-1.medium.com/fit/c/36/36/1*rSGF7OBUjv3Ac2kP_cSjtA.png "The GeekyAnts") [The GeekyAnts](https://blog.geekyants.com/) on medium.com

* Home
* Categories
* Profile
* Notifications
* Bag
